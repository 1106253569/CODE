# Scala语法

| 符号 |       含义       |
| :--: | :--------------: |
| ％d |    十进制数字    |
| ％s |      字符串      |
| ％c |       字符       |
| ％e |    指数浮点数    |
| ％f |      浮点数      |
| ％i |  整数（十进制）  |
| ％o |      八进制      |
| ％u |   无符号十进制   |
| ％x |     十六进制     |
| s" " |  一个模板字符串  |
| f" " | 格式化字符串模板 |

Scala中打印“%”，可以使用两个百分号来转义，如：`println("打印%%")`
